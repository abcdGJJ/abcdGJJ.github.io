  1. 设置双工模式
int g0/0/1
undo negotiation auto (关闭自动协商)
duplex full (全双工)
  1. 配置接口速率
int g0/0/1
undo negotiation auto
speed 100
  1. proxy arp：当主机上没有配置默认网关地址，可以发送一个arp请求使具备proxy arp请求的路由器收到请求后，使用自身的mac地址作为arp请求的回应。

查看arp表：

display arp all
  1. 配置静态arp
arp static 10.1.1.1 0123-4567-89AB
undo arp static 10.1.1.1 0123-4567-89AB

4.1 配置proxy arp (默认情况关闭)

此时PC-1、PC-2、PC-3没有设置网关地址，想要PC-2和PC-3通信需要proxy arp

int g0/0/1
arp-proxy enable
