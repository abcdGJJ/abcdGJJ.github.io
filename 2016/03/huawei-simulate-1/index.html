华为模拟器ensp入门之ensp及VRP基础操作

发表于 2016-03-09 | 更新于 2018-07-23 | 分类于 通信 | 阅读次数：
本文字数： 1.9k | 阅读时长 ≈ 2 分钟

1.进入系统视图：system-view
从系统视图切换到用户视图：quit

2.IP配置：

interface GigabitEthernet 0/0/0
ip address X.X.X.X 24
return

3.修改路由器名称：sysname R1

4.路由器时钟：

clock datetime 12:00:00 2001-1-1
clock timezone BJ add 08:00:00 (东八区)

5.查看路由器信息：

display version (VRP操作系统的版本、型号、启动时间等信息)
disply current-configuration (路由器当前配置)
display interface GigabitEthernet 0/0/0 (查看g0/0/0接口信息)

6.接口上的IP简要信息：display ip interface brief

7.查看路由表：display ip routing-table (用户模式即可)

route flags -- 路由标记
R -- 迭代路由
D -- 该路由下发到FIB表
destination -- 目的网络/主机总数
pre -- 路由优先级
cost -- 路由开销值

8.配置Telnet验证方式 (无用户名方式)：

user-interface vty 0 4
authentication-mode password (默认用户级别为0，参观级)或
set authentication password cipher password
user privilege level 1 (监控级)

telnet连接

telnet X.X.X.X (用户模式下输入)
display users (查看登陆的用户信息)

telnet有用户名方式：

aaa(特权模式下进入aaa模式)
local-user 用户名 password cipher 密码 privilege level 3
local-user 用户名 service-type telnet (配置用户接入的类型为telnet)
user-interface vty 0 4 (退出aaa模式)
authentication-mode aaa

9.Stelnet–secure telnet，走ssh协议

ssh用户使用password方式认证，需要在ssh服务器生成本地RSA密钥，因此生成本地RSA密钥是完成ssh登陆的首要操作。

rsa local-key-pair create
display rsa local-key-pair public (查看本地密钥对中公钥部分信息)
user-interface vty 0 4
authentication-mode aaa
protocol inbound ssh (指定vty界面只支持ssh协议)
aaa
local-user admin password cipher 123456
local-user admin service-type ssh (接入类型ssh)
ssh user admin authentication-type password (退出aaa模式)(新建ssh用户，用户名为admin)
stelnet server enable (启用ssh)

-----------------------配置ssh client----------------------------

ssh client first-time enable 
stelnet X.X.X.X (注意：特权模式)
一路Y下去
display ssh server session  (查看ssh服务端当前会话连接信息)

10.FTP文件操作

  • 配置好客户端和ftp服务器IP地址。并新建一个ftp文件夹。
  • 在路由器上输入
ftp 10.0.2.1 (由于没有设置用户名和密码，所以会让你输入作为初始用户名)
ls
get test.txt (下载到路由器)
put test.txt new.txt (上传到服务器并重命名)
  • 配置路由器为ftp服务器
system-view
ftp server enable
aaa
local-user admin password cipher admin
local-user admin ftp-directory flash (访问目录为ftp)
local-user admin service-type ftp
local-user admin privilege level 15
上传文件测试
