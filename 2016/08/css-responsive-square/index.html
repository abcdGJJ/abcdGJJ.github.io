使用CSS3 vw单位

在CSS3中增加了一系列相对于视窗区域百分比的单位：vw、vh。

其中vw是相对于视窗区域宽度百分比的单位，1vw = 1% viewport width。vh是相对于视窗区域高度百分比的单位，1vh = 1% viewport height。使用这个单位可以很方便的做出自适应正方形。
