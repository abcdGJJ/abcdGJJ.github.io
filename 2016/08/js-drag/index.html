原理

  1. 鼠标按下：状态 = 1；记录鼠标的X和Y坐标；记录元素的X和Y偏移值
  2. 鼠标在元素上移动：若状态 = 0，什么也不做；若状态为1，元素的新X的偏移量 = X2 - X1 + X(鼠标按下时的元素偏移)，新Y偏移量 = Y2 - Y1 + Y
  3. 鼠标放开，状态 = 0
