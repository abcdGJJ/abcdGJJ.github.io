PHP在表格中输出多维数组

发表于 2015-07-27 | 更新于 2018-07-23 | 分类于 技术分享 | 阅读次数：
本文字数： 880 | 阅读时长 ≈ 1 分钟

一个demo:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <?php 
        $student = array
        (
            array('id'=>1,'name'=>'jack','email'=>'aa@qq.com'),
            array('id'=>2,'name'=>'mary','email'=>'aa@qq.com'),
            array('id'=>3,'name'=>'bill','email'=>'aa@qq.com'),
        );
    ?>
    <table border="1">
        <?php foreach ($student as $item): ?>
            <tr>
                <td><?=$item['id']?></td>
                <td><?=$item['name']?></td>
                <td><?=$item['email']?></td>
            </tr>
        <?php endforeach; ?>
    </table>
</body>
</html>

注意：需要php主机支持短标签或php 5.4及以上才可用

TCP–可靠性传输控制（面向连接的可靠传输）

1.使用范围：可靠性较差的网络，比如Internet，传输速度较慢
2.三次握手：

客户机触发服务器，向服务器发出syn=1，ack=0信号
服务器向客户机发送ack=1，syn=1的确认信号（可分步，即形成四次握手）
客户机回应服务器回应syn=0，ack=1信号

3.TCP滑动窗口与确认机制
滑动窗口

发一个数据包确认一个，可靠性高，效率低
发三个数据包确认一个，可靠性低，效率高
滑动窗口确认方式：接收号+1
