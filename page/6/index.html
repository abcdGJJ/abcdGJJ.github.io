本文献给第一次安装git的人

  1. 首先配置个人用户名称和电子邮件地址。这两条配置很重要，每次 Git 提交时都会引用这两条信息，说明是谁提交了更新，所以会随更新内容一起被永久纳入历史记录

    git config --global user.name "yourname"
git config --global user.email youremail

  2. 查看配置信息git config --list，如果出现刚才的用户名和邮箱则说明配置成功。

1.什么是RIP？

RIP–应用较早的、使用较普遍的内部网关协议，典型的矢量距离协议。

2.RIP报文结构

  1. IP首部：包含源地址和目标地址IP。源IP地址为始发路由更新消息的路由器。目标IP版本1为255.255.255.255（广播地址），版本2为224.0.0.9（组播地址）

  2. 源端口和目的端口都为UDP端口，端口号520

  3. Command：RIP消息类型。路由更新请求为1，路由响应请求为2

http–超文本传输协议(hypertext transfer protocol)

1.默认是80端口
2.两个版本

  • 1.0

代理服务器常采用1.0

个人PC采用1.1
