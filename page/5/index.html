最近在看介绍ipv6的书，把书中的重点内容纪录下来，以便日后查看。

ipv6-literal.net域名用“-”替代“:”

eg: 2001:1158:db58::64ef的ipv6.literal.net域名是：2001-1158-db58–64ef

在访问ipv6资源时只能用literal.net域名，因为UNC（通用命名规则）不支持冒号。

eg: 2001-1158-db58–64ef.literal.net\windows

ipv6地址作用于全局，本地链路ipv6地址作用于本子网。

查看路由表的命令：

ipv6 route table

1.OSPF（open shortest path first 开放式最短路径优先）协议：一个内部网关协议，基于链路状态，支持VLSM、快速收敛、使用组播地址（224.0.0.5和224.0.0.6）完成路由更新。

2.链路状态路由和矢量路由（RIP）的区别：

矢量路由只公告自己的路由表，并不关心路由记录是否良好，而且距离也有限制。收敛较慢
链路状态路由不受连接数目的影响，收链快。关心路由状态是否良好
