本文献给第一次安装git的人

  1. 首先配置个人用户名称和电子邮件地址。这两条配置很重要，每次 Git 提交时都会引用这两条信息，说明是谁提交了更新，所以会随更新内容一起被永久纳入历史记录

    git config --global user.name "yourname"
git config --global user.email youremail

  2. 查看配置信息git config --list，如果出现刚才的用户名和邮箱则说明配置成功。

git基础

  1. 克隆git仓库
    git clone https://github.com/ejci/favico.js.git

    （clone后面的参数为该仓库在github上的地址）
  2. 新建git仓库并提交（首先在github建立一个远程仓库）
    git init
git add .
git commit -m "描述"
git remote add origin url
git push -u origin master

git init –初始化，在当前目录建立git仓库

git add .–选择提交的文件

git commit -m “”–对提交描述

git remote add origin url–添加远程主机地址，url即刚才在github建立的仓库的地址

git push -u origin master–提交到github
